# Greedy-Technique
Knapsack Problem

## Overview

This project was created for the CST-201 course at Grand Canyon University. The program demonstrates two approaches for solving the Knapsack Problem:

- Dynamic Programming
- Greedy Technique

The program also solves the bounded knapsack problem by considering the available quantity of each item and compares the performance of both algorithms.

---

## Features

- Solves the original 0/1 Knapsack Problem
- Solves the Bounded Knapsack Problem
- Calculates the maximum value for each algorithm
- Displays the number of steps performed by each algorithm
- Displays the time complexity of each algorithm
- Compares Dynamic Programming and Greedy approaches

|-----------|-----------------|
